# Isitnearme
Finds direction of people around based on probe requests RSSI blockage to create wifi radar. Has compass and web interface, and uses Wigle to import local networks..
<br>
• Uses esp8266 continuously connected to phone for serial communications, gathering requests to push wifi data into web app, which can be run via the index.html file.
<br>
• Requires flashing the esp8266 with the .cpp code, or you can copy and paste into Arduino IDE, or build a .bin file.
